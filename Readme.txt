This is a Simulator of urban simulation for Land use Planning.
Unzip into directory with CD++ tool.
You can open urban Simulation.doc file for more information 
about the specifications and implementation of the system.

# To run the simulation using CD++ you should include these files:
	# Model file: population.ma
	# Input file: Pop_input.val
	# Draw file: PopulationDRW.drw
	# Log file: populationLOG.log
	# Batch file: population.bat
	# Pal file: Population.pal
